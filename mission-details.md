## [Project C] AI 기반 소셜 미디어 콘텐츠 자동 생성

### 1. 미션 소개

인스타그램 캡션을 쓰고 나면 같은 내용으로 블로그 글을 다시 쓰고 또 트위터(X)용으로 맞춰 줄여야 합니다.

같은 주제를 세 번 반복하는 셈인데 매번 플랫폼마다 톤과 포맷을 바꿔가며 작성하다 보면 정작 기획에 쓸 시간이 사라집니다.

이 반복 작업을 자동화하면 어떻게 될까요?

소셜 미디어 마케팅에서 가장 시간이 많이 드는 작업이 바로 하나의 주제를 여러 플랫폼에 맞게 변환하는 일입니다.

인스타그램은 짧은 캡션과 해시태그를 요구하고 블로그는 긴 본문과 소제목 구조가 필요하며 트위터(X)는 특정 글자 수 제한 안에서 핵심을 전달해야 합니다.

이번 미션에서는 하나의 주제를 입력하면 여러 소셜 미디어 플랫폼에 맞는 텍스트와 이미지를 자동으로 생성하는 워크플로우를 팀 단위로 구축합니다.

생성형 AI 툴로 플랫폼별 텍스트를 생성하고 이미지 생성 AI 툴로 대표 이미지를 만든 뒤 결과물을 한곳에 저장하는 전체 파이프라인을 설계하는 것이 핵심입니다.

이 과정을 마치면 플랫폼별 특성을 반영한 맞춤형 콘텐츠를 동시에 생성하는 자동화 시스템을 직접 운영해 본 경험을 갖추게 됩니다.

여기서 익힌 멀티채널 콘텐츠 자동화 설계 역량은 이후 대규모 마케팅 캠페인 운영이나 콘텐츠 프로덕션 파이프라인 구축으로 확장해 나가는 든든한 기반이 될 것입니다.


### 2. 최종 결과물

다음 3가지 산출물을 포함한 AI 기반 소셜 미디어 콘텐츠 자동 생성 시스템을 팀 단위로 완성한다.

1. 자동화 워크플로우

- 주제 입력 시 플랫폼별 콘텐츠가 자동 생성되는 워크플로우

- 워크플로우 구조를 보여주는 스크린샷

- 각 단계별 역할과 연결 구조를 설명하는 문서

2. 플랫폼별 콘텐츠 결과물

- 최소 3개 플랫폼(예: 인스타그램, 블로그, 트위터)에 맞는 텍스트

- 콘텐츠에 활용할 대표 이미지 1장 이상

- 결과물이 저장된 노션 등 협업 도구 페이지

- 협업 도구 내 프로젝트 개요, 팀 구성원 역할, 개인별 작업 요약 포함

3. 프롬프트 템플릿 문서

- 플랫폼별 프롬프트 설계 내용

- 프롬프트 수정 과정 기록(초안 → 수정 → 최종)


### 3. 과제 목표

이 과제를 마친 후, 학습자는 아래를 스스로 설명할 수 있어야 한다.

- 하나의 주제에서 플랫폼별 특성에 맞는 콘텐츠를 자동 생성하는 워크플로우를 설명할 수 있다.

- 생성형 AI 툴과 이미지 생성 AI 툴을 연동하여 텍스트+이미지 콘텐츠를 자동화하는 방법을 설명할 수 있다.

- 콘텐츠 포맷(글자 수, 해시태그, 톤앤매너 등)을 고려한 프롬프트 설계 원리를 설명할 수 있다.


### 4. 기능 요구 사항

다음 요구사항을 모두 만족해야 한다.

1. 입력 및 트리거

- 주제 또는 키워드를 입력받는 방식을 구현한다(폼, 수동 입력 등).

- 입력 시 자동화 워크플로우가 트리거되어야 한다.

2. 플랫폼별 텍스트 생성

- 생성형 AI 툴을 활용하여 최소 3개 플랫폼에 맞는 텍스트를 생성한다.

- 각 플랫폼의 특성을 반영해야 한다.

	- 예: 인스타그램 – 짧은 캡션 + 해시태그 10개 내외

	- 예: 블로그 – 500자 이상 본문 + 소제목 구조

	- 예: X(구 트위터) – 280자 이내 요약

- 톤앤매너가 플랫폼별로 적절하게 조정되어야 한다.

3. 이미지 생성

- 이미지 생성 AI 툴을 활용하여 콘텐츠 주제에 맞는 대표 이미지를 생성한다.

- 생성된 이미지는 소셜 미디어에 활용 가능한 품질이어야 한다.

4. 결과 저장

- 생성된 텍스트와 이미지를 노션 등 협업 도구에 자동 저장한다.

- 플랫폼별로 구분되어 저장되어야 한다.


### 5. 보너스 과제

1. 보너스 1 – A/B 테스트 진행

- 같은 주제로 톤앤매너가 다른 2가지 버전의 콘텐츠를 생성한다(예: 친근한 버전 vs 전문적인 버전).

- 플랫폼과 주제 등 다양한 변수별로 어떤 콘텐츠를 만드는 것이 효과적인지 결론을 정리한다.


### 6. 제약 사항

- 금지 사항

	- 저작권 등의 문제가 있는 콘텐츠를 실제 소셜 미디어에 무단 게시하는 행위

	- 타인의 콘텐츠를 복사하여 제출하는 행위
