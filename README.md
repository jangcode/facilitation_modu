# Study Note: Vibe Coding (바이브 코딩)

> 바이브 코딩(Vibe Coding) 교육을 위한 사례 연구 및 학습 자료 제공을 목표로 하는 프로젝트입니다.

---

## 📌 프로젝트 소개 (About the Project)

이 프로젝트는 최신 AI 기술과 협업 방식을 활용하여, 바이브 코딩 교육의 효과적인 사례 연구와 실무 학습 자료를 구축하고 제공합니다.
사용자는 일상적인 환경(모바일 기기 등)에서 손쉽게 이슈를 발제하고, 인공지능 에이전트(Jules)와의 원격 협업을 통해 프로젝트를 지속적으로 업데이트해 나가는 현대적인 개발 워크플로우를 직접 경험할 수 있습니다.

---

## ⚙️ 워크플로우 (How It Works)

이 프로젝트는 다음과 같은 유기적인 원격 자동화 협업 흐름으로 작동합니다.

```
[사용자 휴대폰 (GitHub 앱)] ── 이슈 등록 (Assign to @jules) ──> [Jules 에이전트 (원격 작업 및 구현)]
                                                                           │
                                                                   PR (Pull Request) 등록
                                                                           │
                                                                           ▼
[Vercel (자동 호스팅 & 배포)] <── PR 승인 (사용자 확인 및 머지) ───────────┘
```

1. **이슈 등록**: 사용자는 휴대폰의 깃헙(GitHub) 앱을 이용하여 작업이 필요한 내용을 이슈로 작성하고 에이전트 `jules`에게 등록합니다.
2. **원격 구현**: 에이전트 `jules`가 이슈를 인지하고 가상 환경에서 소스코드 분석, 구현, 테스트를 직접 원격 수행합니다.
3. **PR 등록**: `jules`가 구현을 마친 후 깃헙에 자동으로 PR(Pull Request)을 등록합니다.
4. **승인 및 업데이트**: 사용자가 PR의 변경 사항을 확인하고 승인(Approve & Merge)하면 코드가 최종 업데이트됩니다.

---

## 🚀 배포 및 호스팅 (Hosting)

이 프로젝트는 최신 웹 프레임워크와의 유연한 연동을 지원하며, 공개될 사이트는 **[Vercel](https://vercel.com/)**에 연동되어 호스팅됩니다.
- 코드의 변경 사항이 메인 브랜치에 반영되는 즉시 실시간(Continuous Delivery)으로 자동 빌드 및 배포가 완료됩니다.

---

## 💡 주요 특징 (Features)
- **모바일 친화성**: 장소에 구애받지 않고 모바일 GitHub 앱을 통해 즉시 프로젝트 유지보수 및 피드백 가능.
- **AI 주도 개발(AI-Driven Development)**: 개발 환경 세팅부터 테스트, 배포 전 단계까지 인공지능 에이전트와 완벽한 시너지 창출.
- **지속적이고 빠른 배포**: Vercel의 서버리스 아키텍처를 통한 최적의 사이트 가속 및 확장성 확보.
