# 미션 B1-2: GenAI 기초 2 - 멀티모달 콘텐츠 제작 가이드

본 디렉토리는 AI 생성형 도구를 활용하여 브랜드 광고 파이프라인을 설계하고, 기획서(스토리보드) 및 관련 산출물을 작성하는 실습 공간입니다.

## 1. 과제 수행 절차
1. **요구사항 분석**: `mission-details.md` 분석 및 `required_mission.md` 작성.
2. **브랜드 기획 및 스토리보드**: 사이버펑크 패션 브랜드 'NeoSeoul'을 테마로 한 상세 씬별 기획 작성 (`storyboard.md`).
3. **도구 선정 및 프롬프트 파이프라인 구성**: 이미지/비디오/오디오 AI 생성기의 속성을 고려한 프롬프트 정의 및 개선 로그 기록.
4. **결과 보고서 작성**: 동료 평가 질문에 답변하고 파이프라인 흐름을 기술한 보고서 작성 (`goal_report.md`).
