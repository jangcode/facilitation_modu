# LLM 모델 비교·선정 보고서

## 1. 비교 목적
회의록 요약 및 업무 자동화 파이프라인 구축을 위해, 각 LLM 모델의 성능을 정량적/정성적 기준에 따라 비교 분석하고 업무에 가장 적합한 최적의 모델을 선정하고자 합니다.

## 2. 사용 환경 및 조건 기록 (재현성)
- **비교 분석 수행일**: 2026-06-11
- **비교 대상 모델 및 사용 환경**:
  1. **GPT-5.5 Instant**: 웹 인터페이스 (ChatGPT free) / Temperature: 기본값
  2. **Claude 4.6 Sonnet**: 웹 인터페이스 (Claude free) / Temperature: 기본값
  3. **Gemini 3.5 Flash**: 웹 인터페이스 (Gemini free) / Temperature: 기본값

## 3. 비교에 사용한 동일 입력 (테스트 프롬프트)
```markdown
[업무 과업] 회의록 요약

[원문]
(B1-1/회의록.md의 '수입 주류 진품인증 마그네틱 씰 공식 인정 관련 협의 회의록' 전문 입력)

[원하는 결과] 
- 최종 결론 요약 (3문장 이내)
- 결정사항 (불릿 형태)
- Action Items (담당자, 기한, 세부 작업 포함)

[톤] 격식 있고 간결한 사내 공유용 보고서 톤
[금지 사항] 추측성 문장 작성 금지, 회의록에 등장하지 않은 인물이나 날짜 언급 금지.
```

## 4. 평가 축 및 점수표
평가는 각 항목당 1점(매우 미흡) ~ 5점(매우 우수) 척도로 진행하였습니다.

| 평가 축 (Criteria) | 설명 | GPT-5.5 Instant | Claude 4.6 Sonnet | Gemini 3.5 Flash |
| :--- | :--- | :---: | :---: | :---: |
| **1) 정확성 (사실 반영)** | 회의록 내 사실(날짜, 인물명, 직책 등) 관계를 왜곡이나 누락 없이 정확하게 파악하는지 | 5 | 5 | 5 |
| **2) 한국어 자연스러움** | 번역 투나 어색한 한자어를 배제하고 자연스러운 비즈니스 한국어로 요약하는지 | 4 | 5 | 4 |
| **3) 문맥 유지력** | 대화 흐름 속에서 지시 사항 및 제약 조건을 일관되게 기억하고 반영하는지 | 4 | 5 | 4 |
| **4) 형식 준수성** | 요구한 출력 규격(템플릿 구조, 불릿 등) 및 금지 규칙을 완전하게 지키는지 | 4.5 | 5 | 3 |
| **평균 점수** | | **4.38** | **5.0** | **4.0** |

## 5. 모델별 결과 요약 및 평가 근거

### GPT-5.5 Instant
- **정확성 & 형식 준수**: 회의록 내 사실 관계를 정확히 인지하고, 표 형식을 활용해 Action Items를 깔끔하게 작성하였습니다. 다만 요약 부분에 번호를 매겨 제시하였고, 시작 부분에 불필요한 도입부 안내 멘트("다음은 첨부된 회의록...")가 포함되어 보고서 문서로서의 완성도가 다소 저해되었습니다.
- **한국어 자연스러움**: 문장 자체는 매끄러운 보고서용 어조를 구현하였으나, 챗봇 특유의 메타 멘트가 남아 있어 자연스러움 면에서 감점 요인이 존재합니다.
- **문맥 유지력**: 금지 조건(추측 금지, 미등장 인물/날짜 언급 금지)을 위배하지 않고 안정적으로 문맥을 유지했습니다.

### Claude 4.6 Sonnet
- **정확성 & 한국어 자연스러움**: 요구 조건인 '격식 있고 간결한 사내 공유용 보고서 톤'을 완벽하게 만족합니다. 비즈니스 한국어 표현력이 3개 모델 중 가장 탁월하며, 문맥의 흐름이 매끄럽고 정교합니다.
- **형식 준수성**: 3문장 이내 요약, 불릿 형태의 결정사항을 완벽히 이행했으며, Action Items를 가독성이 뛰어난 표(Table) 형식으로 깔끔하게 구성하였습니다. 특히 회의록 내에 기한이 명확하지 않은 항목을 안전하게 '미정'으로 표기하고 하단에 주석을 덧붙이는 꼼꼼함을 보였습니다.
- **문맥 유지력**: 제약 사항 및 사실 관계 통제력이 매우 우수하여 왜곡이나 가공 없는 결과물을 제공합니다.

### Gemini 3.5 Flash
- **정확성 & 문맥 유지력**: 회의 내용과 인물명, 역할 등을 누락 없이 정확하게 추출하였으며, 오정보나 추측성 내용을 작성하지 않아 기초 사실 반영 능력이 뛰어납니다.
- **한국어 자연스러움**: 보고서에 적합한 격식 있는 문어체가 아닌 존댓말 구어체(~밝혔습니다, ~예정입니다)로 작성되어 사내 보고서 톤으로는 다소 격이 떨어집니다.
- **형식 준수성**: 줄바꿈 오류(Double spacing)로 인해 마크다운 출력 포맷이 매우 불규칙하고 엉성하게 생성되었으며, Action Items가 표 형식이 아닌 단순 텍스트 나열로 되어 가독성이 부족합니다.

## 6. 최종 선정 결론
본 '회의록 요약 및 정보 관리' 과업을 수행할 최적의 모델로 **Claude 4.6 Sonnet**을 선정합니다.

**선정 근거**:
1. 사내 공유 보고서에 가장 부합하는 격식 있고 깔끔한 비즈니스 문어체 톤을 완벽하게 구사하여 추가적인 수정 공수를 최소화합니다.
2. Action Items를 표 형식으로 정돈하고, 가독성을 고려한 마크다운 포맷을 오차 없이 일관되게 출력하여 템플릿 준수 성능이 가장 뛰어납니다.
3. 챗봇 고유의 불필요한 메타 멘트(안내 문구)를 배제하고 지시사항(제약 조건)만을 깔끔하게 전달하는 등 지시 통제 능력이 독보적입니다.
