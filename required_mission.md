## 과제명

[Project C] AI 기반 소셜 미디어 콘텐츠 자동 생성

## 요구사항 정리

1. **입력 및 트리거**: 
   - 주제 또는 키워드를 입력받는 방식 구현(폼, 수동 입력 등).
   - 입력 시 자동화 워크플로우 트리거.
2. **플랫폼별 텍스트 생성**: 
   - 생성형 AI 툴을 활용해 최소 3개 플랫폼(인스타그램, 블로그, X 등)에 맞는 텍스트 생성.
   - 플랫폼별 특성(글자 수 제한, 해시태그, 구조 등)과 톤앤매너 반영.
3. **이미지 생성**: 
   - 이미지 생성 AI 툴을 활용하여 소셜 미디어에 활용 가능한 품질의 콘텐츠 주제에 맞는 대표 이미지 생성.
4. **결과 저장**: 
   - 생성된 텍스트와 이미지를 노션 등 협업 도구에 자동 저장 (플랫폼별 구분).
5. **(보너스) A/B 테스트 진행**: 
   - 동일 주제로 톤앤매너가 다른 2가지 버전 콘텐츠 생성 후 효과 비교.

## 산출물 목록

1. **자동화 워크플로우**:
   - 주제 입력 시 플랫폼별 콘텐츠가 자동 생성되는 워크플로우
   - 워크플로우 구조 스크린샷
   - 각 단계별 역할과 연결 구조 설명 문서
2. **플랫폼별 콘텐츠 결과물**:
   - 최소 3개 플랫폼용 맞춤형 텍스트
   - 대표 이미지 1장 이상
   - 결과물이 저장된 노션(등 협업 도구) 페이지 링크/스크린샷
   - 프로젝트 개요, 팀원 역할, 개인 작업 요약
3. **프롬프트 템플릿 문서**:
   - 플랫폼별 프롬프트 설계 내용
   - 프롬프트 수정 과정 기록 (초안 → 수정 → 최종)
