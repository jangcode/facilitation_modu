## 과제명
[Project A] AI 기반 UI/UX 디자인 시안 제작

## 요구사항 정리
1. **이미지 생성**
   - 이미지 생성 AI 도구(Midjourney, DALL-E, Stable Diffusion 등) 최소 1개 이상 사용.
   - 최소 3개 이상의 화면(메인 페이지, 목록 페이지, 상세 페이지 등) 생성.
   - 생성된 UI 디자인은 모바일(9:16) 또는 데스크탑(16:9)의 일반적인 화면 비율 준수.
2. **프롬프트 최적화**
   - 프롬프트 개선 과정(초안 → 수정 → 최종) 문서로 기록.
   - 각 수정 단계에서 변경한 이유와 결과 차이 기록.
3. **이미지 후가공**
   - AI가 생성한 이미지 속 뭉개진 글자나 부자연스러운 요소 수정.
   - 필요시 인페인팅, 아웃페인팅, 업스케일링 등의 기능 활용.
4. **프로토타입 제작**
   - Figma 등 디자인 도구를 사용하여 생성된 이미지 배치.
   - 화면 간 이동 흐름을 나타내는 클릭 가능 영역(Hotspot) 또는 화면 전환 표시.
5. **제약 사항**
   - 저작권 문제가 해결된 AI 도구 사용 및 툴 이름 반드시 명시.
   - 디자인 편집 및 배치는 Figma 사용 권장 (다른 도구 활용 시 이유와 도구명 명시).
   - 레퍼런스 활용 시 출처 필수 기록 (타인 작업물 단순 캡처 제출 금지).
   - 생성된 이미지 속 부자연스러운 요소 수정 없이 제출 불가.

## 산출물 목록
1. **UI 디자인 이미지 (3장 이상)**
   - 서비스 핵심 기능을 표현한 화면 (PNG 또는 JPG)
   - 화면 역할 구분 (메인, 목록, 상세 등)
   - 텍스트 깨짐 및 부자연스러운 요소가 수정된 상태
2. **작업 로그 문서 (1개)**
   - 디자인 생성에 사용한 프롬프트 기록
   - 프롬프트 수정 과정 (초안 → 수정 → 최종)
   - 수정 단계별 변경 이유 및 결과 차이 기록
3. **Figma 프로젝트 (1개)**
   - 생성된 이미지를 배치하여 화면 흐름을 구성한 프로젝트 링크
   - 클릭 가능한 영역(Hotspot) 또는 화면 전환 흐름 표시
