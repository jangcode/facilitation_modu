# [Project A] 작업 로그

## 1. 이미지 생성 AI 도구
- **사용 도구**: Stitch (AI 기반 UI/UX 디자인 도구)
- **주제**: 여행 및 숙박 예약 모바일 앱
- **선정 이유**: DALL-E 등 범용 이미지 생성기를 사용할 경우 발생하는 UI 내 텍스트 깨짐 및 레이아웃 불일치 문제를 해결하기 위해, UI 구조 생성과 텍스트 제어가 동시에 가능하고 화면 연결(Hotspot)까지 바로 진행할 수 있는 특화 도구인 Stitch를 사용함.

## 2. 프롬프트 개선 과정 및 결과

### 화면 1: 메인/검색 화면 (Home Screen)
| 단계 | 프롬프트 | 결과 및 개선점 |
|------|----------|----------------|
| **초안** | `travel app home screen, blue theme` | 너무 단순하고 모바일 레이아웃에 대한 개념이 부족하여 비율이 맞지 않음. |
| **수정** | `Mobile app UI design for travel booking, home screen, blue and white theme, search bar` | 디자인은 나아졌으나 하단 네비게이션이 없고 텍스트가 깨지거나 외계어로 나옴. |
| **최종** | `Mobile app UI design for a travel booking app, home screen. Clean minimal style, blue and white theme. Shows a search bar at the top with text 'Where to?', destination recommendation cards in the middle, and a bottom navigation bar with Home, Saved, Profile.` | 프롬프트에 'Where to?', 'Home', 'Saved', 'Profile' 등 구체적 텍스트를 명시하여 외계어 텍스트 문제를 예방하고, 일관된 미니멀 스타일의 검색바와 네비게이션이 성공적으로 생성됨. |

### 화면 2: 검색 결과 화면 (Search Results Screen)
| 단계 | 프롬프트 | 결과 및 개선점 |
|------|----------|----------------|
| **초안** | `travel app search results, hotel list` | 리스트가 너무 빽빽하고 필터 버튼이 없음. |
| **수정** | `Mobile app UI design for travel app search results, list of hotels, blue and white theme` | 호텔 리스트는 잘 나왔으나 UI가 구식 느낌이 나며 통화 기호나 필터 버튼 이름이 부정확함. |
| **최종** | `Mobile app UI design for a travel booking app, search results screen. Clean minimal style, blue and white theme. Shows a list of hotels with thumbnail images, prices (e.g. $120/night), star ratings, and a filter button at the top labeled 'Filters'.` | 구체적인 가격 예시('$120/night')와 명확한 레이블('Filters')을 포함하여 텍스트 깨짐을 방지하고, 고품질 썸네일과 별점을 포함한 현대적인 리스트 UI를 도출함. |

### 화면 3: 상세 페이지 (Detail Screen)
| 단계 | 프롬프트 | 결과 및 개선점 |
|------|----------|----------------|
| **초안** | `travel app hotel detail page` | 단순히 방 이미지만 크게 나오고 예약 버튼이 명확하지 않음. |
| **수정** | `Mobile app UI design for hotel detail, large cover image, book now button` | 필요한 요소는 있으나 레이아웃 일관성이 떨어지고 본문이 더미 텍스트로 채워짐. |
| **최종** | `Mobile app UI design for a travel booking app, hotel detail screen. Clean minimal style, blue and white theme. Shows a large cover image of a hotel room at the top, hotel name, review summary, description text, and a prominent 'Book Now' button at the bottom.` | Stitch를 통해 퀄리티 높은 커버 이미지, 리뷰 요약, 그리고 하단에 명확하게 강조된 'Book Now' 버튼이 배치된 모바일 화면을 완벽히 생성함. |

## 3. 이미지 후가공 및 프로토타입 연결
- **과정**: Stitch 내에서 생성된 세 가지 화면(Home, Search Results, Detail)을 하나의 프로젝트 워크스페이스에 통합.
- **후가공 및 검수**: Stitch는 생성 시점에서 텍스트를 컴포넌트로 구조화하여 반환하므로, 기존 이미지 생성 AI의 "텍스트 깨짐 현상"을 효율적으로 회피함. 프롬프트로 명시되지 않아 어색하게 생성된 일부 레이블은 에디터 상호작용으로 직접 실제 서비스 문구로 교체함.
- **프로토타이핑**: 별도의 Figma 이동 없이 Stitch 내부의 네비게이션(Hotspot) 연결 기능을 활용하여, Home → Search Results → Detail 로 이어지는 클릭 가능한 화면 전환 흐름을 구성, 미션 요구사항을 성공적으로 충족함.
