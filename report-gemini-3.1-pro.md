### 💻 3. 파이썬으로 직접 분석해 보기 (보너스!)

데이터가 더 많아지면 눈으로 확인하기 어렵겠죠? 파이썬의 `pandas` 라이브러리를 사용하면 이런 데이터를 아주 쉽게 분석할 수 있어요. 아래 코드를 한 번 살펴보세요!

```python
import pandas as pd

# 1. CSV 파일 불러오기 (데이터를 표 형태로 가져옵니다)
df = pd.read_csv('cost-report.csv')

# 2. 서비스별 총 비용 계산하기
# 'service' 컬럼을 기준으로 그룹을 묶고, 'cost_usd'의 합계를 구합니다.
total_cost_by_service = df.groupby('service')['cost_usd'].sum().reset_index()

# 3. 비용이 높은 순서대로 정렬하기
total_cost_by_service = total_cost_by_service.sort_values(by='cost_usd', ascending=False)

print("=== 서비스별 총 사용 요금 ===")
print(total_cost_by_service)
```

**📝 코드 설명:**
* `pd.read_csv()`: 엑셀이나 CSV 파일을 파이썬으로 읽어오는 마법 같은 함수예요.
* `groupby('service')`: 같은 서비스(EC2, S3 등)끼리 데이터를 묶어줍니다.
* `sum()`: 묶인 데이터들의 요금을 모두 더해줍니다.
